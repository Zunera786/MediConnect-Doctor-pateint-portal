MediConnect Enhanced UI
-----------------------
This package contains the enhanced front-end UI for MediConnect:
- Brand navbar with 'MediConnect' shown prominently
- Enhanced Home, About, Help, Contact pages
- Separate, modern login pages: patientlogin.php, doctorlogin.php, adminlogin.php
- Separate registration pages: patientregister.php, doctorregister.php (linked from register.php)
- Empty dashboards and admin management pages (placeholders ready for backend integration)
- assets/css/style.css contains the theme and styles

Stack: Plain PHP + HTML, ready for MySQL integration.
